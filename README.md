# hubot-asaichi

A Hubot script that returns asaichi schedule

## Installation

    $ npm install https://github.com/bouzuya/hubot-asaichi/archive/master.tar.gz

or

    $ npm install https://github.com/bouzuya/hubot-asaichi/archive/{VERSION}.tar.gz
